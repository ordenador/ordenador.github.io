## Getting Started 🛠️

### Prerequisites

- [Node.js](package.json)
- [Python](requirements.txt)


### Essential Commands

```bash
# Install dependencies
make install

# Build for production
make build

# Start development server
make start

# Generate PDF version
make generate-pdf
```
